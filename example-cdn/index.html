Bootstrap AI CDN Example

This example shows how to use Bootstrap AI components directly from CDN without any build step.

Note: This example uses CDN links that will work once the package is published to npm. For now, use the local example.
<script type="importmap"> { "imports": { "@bootstrap-ai/ui": "https://cdn.jsdelivr.net/npm/@bootstrap-ai/ui@latest/dist/index.js", "lit": "https://cdn.jsdelivr.net/npm/lit@latest/index.js", "lit/": "https://cdn.jsdelivr.net/npm/lit@latest/" } } </script>
